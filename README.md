# Room Runner
> Room Runner ist das erste Spiel, das ich fertiggestellt habe. Es handelt sich um einen Dungeon Crawler, in dem man sich durch verschiedene Karten bewegt, Gegner besiegt und mit dem verdienten Geld neue Karten freischaltet.
---
## Spielprinzip
- Maps erkunden
- Besiege Gegner
- Geld Einsameln
- Neue Maps kaufen

## Steuerung
- Bewegen: **w a s d**/**Pfeiltsaten**
- Angriff: **rechtsklick**

## Installation
1. Geh zum GitHub-Repository
2. Klicke auf der Seite auf Releases.
3. Installiere die ZIP-Datei
4. Entpacke den ZIP-Ordner
5. Starte die Room.runner.zip
